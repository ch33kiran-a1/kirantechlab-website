Tools & Technologies Used

  • GitHub
  • Azure DevOps Pipelines

  • AWS EC2

Infrastructure

  • AWS EC2 Ubuntu Server
  • Elastic IP for static public access
  • Nginx configured as web server
  • Domain configured via DNS
  • SSL certificate using Let's Encrypt

Deployment is fully automated using Azure DevOps CI/CD.
